Mahu 文档站点默认集成了 `nuxt-llms` 模块，会自动生成 `/llms.txt` 与 `/llms-full.txt` 两个文件，供大模型直接索引文档内容。

## 默认值

`/llms.txt` 文件基于以下默认值生成：

- `domain` → 根据部署平台自动计算（或通过环境变量 `NUXT_SITE_URL` 指定）
- `title` → 从 `package.json` 的 `name` 字段读取
- `description` → 从 `package.json` 的 `description` 字段读取

## 自定义配置

可在 `nuxt.config.ts` 中覆盖 LLM 索引信息：

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  llms: {
    domain: 'https://your-docs-url',
    title: 'Mahu 开发文档',
    description: '面向 Mahu 内部开发者的项目总览、架构设计与数据库规范入口。',
    full: {
      title: 'Mahu 开发文档（完整版）',
      description: '面向 Mahu 内部开发者的项目总览、架构设计与数据库规范入口。',
    },
  },
})
```
