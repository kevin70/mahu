## MCP 简介

[Model Context Protocol (MCP)](https://modelcontextprotocol.io/) 是一个开放协议，用于在 AI 应用与外部服务（如文档站点）之间建立标准化连接。

Mahu 文档站点内置了 MCP Server，允许 Claude、Cursor、VS Code 等支持 MCP 的客户端直接读取本文档内容。

### 工作原理

当 AI 工具连接到 MCP Server 后，大模型可以在生成回答时**主动检索文档**：

- 模型根据对话上下文自主决定何时查询文档。
- 每次工具调用发生在生成过程中，允许模型将文档中的实时信息纳入回答。
- 无需用户显式要求——只要问题与文档内容相关，模型即可自动检索。

## 访问 MCP Server

Mahu 文档的 MCP Server 默认挂载在文档站点 URL 的 `/mcp` 路径下：

```
https://<your-docs-url>/mcp
```

## 内置工具

| 工具 | 说明 |
|------|------|
| `list-pages` | 列出所有文档页面（标题、路径、描述），AI 应先调用此工具发现可用内容 |
| `get-page` | 获取指定路径页面的完整 Markdown 内容（参数：`path`，如 `/getting-started/installation`） |

## 在常见工具中接入

### Claude Code

```bash [Terminal]
claude mcp add --transport http mahu-docs https://<your-docs-url>/mcp
```

### Cursor

手动创建或更新项目根目录的 `.cursor/mcp.json`：

```json [.cursor/mcp.json]
{
  "mcpServers": {
    "mahu-docs": {
      "type": "http",
      "url": "https://<your-docs-url>/mcp"
    }
  }
}
```

### Visual Studio Code

手动创建或更新 `.vscode/mcp.json`：

```json [.vscode/mcp.json]
{
  "servers": {
    "mahu-docs": {
      "type": "http",
      "url": "https://<your-docs-url>/mcp"
    }
  }
}
```

### Windsurf

打开 **Settings** > **Windsurf Settings** > **Cascade** > **Manage MCPs** > **View raw config**，添加：

```json [.codeium/windsurf/mcp_config.json]
{
  "mcpServers": {
    "mahu-docs": {
      "type": "http",
      "url": "https://<your-docs-url>/mcp"
    }
  }
}
```

## 关闭 MCP Server（可选）

如需禁用，在 `nuxt.config.ts` 中配置：

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  mcp: {
    enabled: false,
  },
})
```
