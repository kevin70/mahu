## 前置条件

- **JDK 25**
- **Git**
- **Node.js / pnpm**（用于 `docs/` 文档站点）
- **PostgreSQL**（业务开发与本地验证用）

具体版本与本地依赖清单以仓库根目录 `README.md` 为准。

## 拉取代码

```bash [Terminal]
git clone <your-repo-url>
cd mahu
```

## 构建与测试（后端）

Windows PowerShell 推荐使用 `.\gradlew.bat`：

```powershell [Terminal]
.\gradlew.bat build
```

（只跑某个模块的构建/测试也可以按需执行，例如 `:mahu-web:build`、`:mahu-domain:test`。）

## 启动文档站点（docs）

文档站点位于 `docs/`，本地开发启动：

```powershell [Terminal]
cd docs
pnpm install
pnpm dev
```

文档内容主要在 `docs/content/`，修改后会热更新。
