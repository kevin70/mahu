Mahu 是一个以 Java/Helidon SE 为核心的后端工程，仓库内包含多个业务与基础设施模块；同时提供本文档站点，用于沉淀架构规范、数据库约定与 AI 集成方式。

## 技术栈

| 分类 | 技术 |
|------|------|
| 语言 / 运行时 | Java 25 |
| Web 框架 | Helidon SE 4.4 |
| ORM | EBean 17 |
| 数据库 | PostgreSQL 17 |
| 数据库迁移 | Liquibase（格式化 SQL） |
| 对象存储 | MinIO |
| 消息队列 | RabbitMQ |
| 定时任务 | db-scheduler |
| 缓存 | Valkey |
| 依赖注入 | Helidon Service Registry（编译时） |
| 构建工具 | Gradle（多模块） |
| API 规范 | OpenAPI 3.0 |
| 代码格式 | Palantir Java Format |

## 模块一览

仓库以 Gradle 多模块形式组织（以 `settings.gradle` 为准）：

| 模块 | 说明 |
|------|------|
| `mahu-bom` | Java Platform / BOM，统一依赖版本（配合 `gradle/libs.versions.toml`） |
| `mahu-base` | 基础设施与通用工具 |
| `mahu-domain` | 领域层：EBean 实体、配置模型、枚举定义 |
| `mahu-codegen` | 代码生成相关工具 |
| `mahu-db` | Liquibase schema 变更脚本（只做 schema，不写业务逻辑） |
| `mahu-dal` | 数据访问层（Repository 实现，基于 EBean） |
| `mahu-shared` | 跨应用共享的领域/应用服务 |
| `mahu-web` | 通用 HTTP/Web 能力（Helidon WebServer 封装、错误处理） |
| `mahu-admin:mahu-admin-infra` | 管理端基础设施与业务服务层 |
| `mahu-admin:mahu-admin-web` | 管理端 Helidon SE API 应用（基于 OpenAPI 生成） |
| `mahu-task` | 独立定时任务服务（db-scheduler） |

## Java Package 规范

包结构以功能后缀命名，各层职责如下：

| 包名 | 说明 |
|------|------|
| `util` | 工具类 |
| `entity` | 数据实体，与数据库表一一对应 |
| `repository` | 数据库访问对象（类名以 `Repository` 结尾） |
| `service` | 业务服务（类名以 `Service` 结尾） |
| `shared` | 跨模块共享的内部公共业务逻辑 |
| `controller` | REST 接口（类名以 `Controller` 结尾） |
